# SST PRIME — Login Animado

Projeto reformulado do zero em HTML, CSS e JavaScript puro.

## O que foi incluído

- Loading inicial inspirado no modelo de código/login enviado.
- Login animado com círculo de barras neon.
- Senha padrão: `SG393`.
- Dashboard, EPIs, riscos, empresas, frases, 5W2H, NRs e painel.
- Layout responsivo para desktop e celular.
- Tema claro/escuro no botão superior.
- Base `DATA` mantida em `JS/data.js`.

## Como usar

Abra o arquivo `index.html` no navegador.

Para publicar no GitHub Pages, envie todos os arquivos mantendo esta estrutura:

```txt
index.html
CSS/style.css
JS/app.js
JS/data.js
assets/equipe-sst-prime.webp
```
